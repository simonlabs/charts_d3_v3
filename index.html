Comparison

  1. Ext.chart
    1. Comercial license
    2. Ext 3.4 chart bases on Flash
    3. Limited chart types
  2. chart.js
    1. MIT license
    2. It bases on HTML5 canvas
    3. Limited chart types
  3. highcharts.js
    1. Comercial license
    2. It bases on HTML5 SVG
    3. Has 18 chart types
  4. d3.js
    1. BSD license
    2. It bases on HTML5 SVG
    3. The most used open source charting library by far
    4. Highly customizable
    5. Well documented
    6. Large collection of reusable examples
  5. c3.js
    1. MIT license
    2. A library wraps around d3.js
Source: http://socialcompare.com/en/comparison/chart-js-c3-d3-highchart-fusioncharts-google-chart-2i8mwb90
